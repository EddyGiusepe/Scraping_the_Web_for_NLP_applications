## <font color="red">Problem</font>

A extração de dados tradicional com BeautifulSoup segue um padrão familiar: buscar o HTML, inspecionar os elementos nas Ferramentas de Desenvolvedor e escrever seletores CSS para extrair os dados.

Mas os sites não permanecem estáticos. Quando a estrutura HTML muda, seus seletores param de funcionar e você precisa reescrever o código novamente.

## <font color="red">Solução</font>

O `ScrapeGraphAI` usa LLMs para extrair dados de descrições em linguagem natural. Basta descrever o que você deseja em inglês simples, e o LLM descobre a lógica de extração automaticamente.

Principais características:

* Raspadores de código com capacidade de autorreparação que se adaptam quando os sites são redesenhados.

* Saída com segurança de tipos e validação de esquema `Pydantic`.

* Renderização `JavaScript` integrada para sites `React`, `Vue` e `Angular`.

* Extração de dados de várias páginas com `SearchGraph` para tarefas de pesquisa

* Modelos em nuvem ou locais via `OpenAI`, `Anthropic` ou `Ollama`.

Além disso, o `ScrapeGraphAI` é de código aberto! Instale-o com `pip install scrapegraphai`.

Existem outros fluxos de trabalho que podem ser usados ​​para extrair informações de várias páginas, gerar scripts em Python ou até mesmo gerar arquivos de áudio.


| Nome do Pipeline | Descrição |
|---------------|-------------|
| `SmartScraperGraph` | Raspador/scraper de página única que só precisa de um prompt do usuário e uma fonte de entrada. |
| `SearchGraph` | Raspador/scraper de várias páginas que extrai informações dos top `n` resultados de pesquisa de um motor de busca. |
| `SpeechGraph` | Raspador/scraper de página única que extrai informações de um site e gera um arquivo de áudio. |
| `ScriptCreatorGraph` | Raspador/scraper de página única que extrai informações de um site e gera um script em Python. |
| `SmartScraperMultiGraph` | Raspador/scraper de várias páginas que extrai informações de várias páginas dadas um único prompt e uma lista de fontes. |
| `ScriptCreatorMultiGraph` | Raspador/scraper de várias páginas que gera um script em Python para extrair informações de várias páginas e fontes. |
